# WalletButton

> Примечание: В настоящее время WalletButton полагается на стандарт кошельков EIP-1193, но в ближайшем будущем будет поддерживать EIP-6963.

Новый компонент WalletButton помогает dApps с реализациями пользовательских списков кошельков принимать RainbowKit и все его преимущества без обслуживания.

```tsx
import { WalletButton } from '@rainbow-me/rainbowkit';

<WalletButton wallet="rainbow" />
<WalletButton wallet="metamask" />
<WalletButton wallet="coinbase" />
```

Как и ConnectButton, компонент WalletButton.Custom доступен для пользовательских реализаций и стилизации.

```tsx
<WalletButton.Custom wallet="rainbow">
  {({ ready, connect }) => {
    return (
      <button
        type="button"
        disabled={!ready}
        onClick={connect}
      >
        Connect Rainbow
      </button>
    );
  }}
</WalletButton.Custom>
```

Большинство dApps лучше всего обслуживаются использованием ConnectButton. Больше информации об использовании и вариантах применения WalletButton можно найти в документации.
