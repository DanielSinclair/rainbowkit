# 自定義錢包列表

## 自定義錢包列表

您可以從 `'@rainbow-me/rainbowkit/wallets'` 導入個別錢包，並使用 `connectorsForWallets` 函數建立你自己的錢包列表，其中包含所需的連接器。 這種方式您可以完全控制顯示哪些錢包以及它們的顯示順序。

例如，您可以選擇僅顯示Rainbow與一般的後備錢包。

```tsx
import { connectorsForWallets } from '@rainbow-me/rainbowkit';
import {
  rainbowWallet,
  walletConnectWallet,
} from '@rainbow-me/rainbowkit/wallets';

const connectors = connectorsForWallets(
  [
    {
      groupName: 'Recommended',
      wallets: [rainbowWallet, walletConnectWallet],
    },
  ],
  {
    appName: 'My RainbowKit App',
    projectId: 'YOUR_PROJECT_ID',
  }
);
```

您可以然後將您的連接器傳遞給Wagmi的 `createConfig`。

```tsx
import { connectorsForWallets } from '@rainbow-me/rainbowkit';
import { createConfig } from 'wagmi';

const connectors = connectorsForWallets(/* ... */);

const config = createConfig({
  connectors,
  {/* Wagmi config */}
});

const queryClient = new QueryClient();

const App = () => (
  <WagmiProvider config={config}>
    <QueryClientProvider client={queryClient}>
      <RainbowKitProvider {...etc}>
        {/* Your App */}
      </RainbowKitProvider>
    </QueryClientProvider>
  </WagmiProvider>
);
```

### 內建錢包

以下錢包是通過 `wallet` 對象提供的。

#### 一般錢包

以下錢包適用於一般的連接方法，而不是特定應用。 因此，建議始終包含這些錢包。

##### WalletConnect

這是一種後備錢包選項，設計用於尚未由列表中的其他錢包提供的基於WalletConnect的錢包。

**建議始終在列表中包含此錢包，以確保支持所有基於WalletConnect的錢包。**

```tsx
import { walletConnectWallet } from '@rainbow-me/rainbowkit/wallets';
```

##### 注入錢包

這是一種後備錢包選項，設計用於當 `window.ethereum` 存在但尚未由列表中的其他錢包提供的情況。 當不再需要後備錢包或者沒有可用的注入錢包，這個錢包會自動從列表中隱藏。

**建議始終在列表中包含此錢包，以確保支持所有注入的錢包。**

```tsx
import { injectedWallet } from '@rainbow-me/rainbowkit/wallets';
```

##### Safe Wallet

對於支持Safe Apps的dApps，允許用戶容易地連接到他們的Safe vault或多重簽名，包含 `safeWallet` 選項進行自動連接。 Safe選項只會在Safe Wallet瀏覽器環境中出現。

```tsx
import { safeWallet } from '@rainbow-me/rainbowkit/wallets';
```

#### Coinbase

這個錢包連接器支持Coinbase Wallet應用和擴展，以及Web上的Coinbase Smart Wallet。

有一個 `preference` 參數可以控制是否啟用並可供用戶使用Smart Wallet。 基於偏好的行為記錄在[這裡](https://www.smartwallet.dev/sdk/makeWeb3Provider#parameters)。

Smart Wallet將在6月初預設啟用 `all`，無需進一步升級。

開發者可以通過設置 `smartWalletOnly` 並將 `coinbaseWallet` 包含在他們的錢包列表中來使用 `sepolia` 和 `baseSepolia` 鏈測試Smart Wallet，如下所示：

```tsx
import { coinbaseWallet } from '@rainbow-me/rainbowkit/wallets';

// Enable Coinbase Smart Wallet for testing
coinbaseWallet.preference = 'smartWalletOnly';
```

### 範例

#### 排序

以下是一些展示不同錢包、不同順序的示例。

顯示MetaMask以及一般的後備錢包。

```tsx
import { connectorsForWallets } from '@rainbow-me/rainbowkit';
import {
  metaMaskWallet,
  walletConnectWallet,
} from '@rainbow-me/rainbowkit/wallets';

const connectors = connectorsForWallets(
  [
    {
      groupName: 'Recommended',
      wallets: [metaMaskWallet, walletConnectWallet],
    },
  ],
  { appName: 'RainbowKit App', projectId: 'YOUR_PROJECT_ID' },
);
```

顯示Rainbow、MetaMask和Coinbase以及一般的後備錢包。

```tsx
import { connectorsForWallets } from '@rainbow-me/rainbowkit';
import {
  rainbowWallet,
  metaMaskWallet,
  coinbaseWallet,
  walletConnectWallet,
} from '@rainbow-me/rainbowkit/wallets';

const connectors = connectorsForWallets(
  [
    {
      groupName: 'Suggested',
      wallets: [
        rainbowWallet,
        metaMaskWallet,
        coinbaseWallet,
        walletConnectWallet,
      ],
    },
  ],
  { appName: 'RainbowKit App', projectId: 'YOUR_PROJECT_ID' },
);
```

> 提醒：`wallets`數組的順序定義了錢包在UI中顯示的順序。

#### 多個群組

您可以使用 `groupName` 鍵為不同的錢包群組命名。 這很有用，若您希望向用戶傳達您推薦哪些錢包，以及其他可能的錢包。

推薦Rainbow和MetaMask，但也提供Coinbase以及一般的後備錢包。

```tsx
import { connectorsForWallets } from '@rainbow-me/rainbowkit';
import {
  rainbowWallet,
  metaMaskWallet,
  coinbaseWallet,
  walletConnectWallet,
} from '@rainbow-me/rainbowkit/wallets';

const connectors = connectorsForWallets(
  [
    {
      groupName: 'Recommended',
      wallets: [rainbowWallet, metaMaskWallet],
    },
    {
      groupName: 'Others',
      wallets: [coinbaseWallet, walletConnectWallet],
    },
  ],
  { appName: 'RainbowKit App', projectId: 'YOUR_PROJECT_ID' },
);
```
