# Cool 模式

## 啟用 Cool 模式 🌈🌈🌈

你可以通過 `RainbowKitProvider` 上的 `coolMode` 屬性啟用 Cool 模式。

```tsx
import { RainbowKitProvider } from '@rainbow-me/rainbowkit';

const App = () => (
  <RainbowKitProvider coolMode {...etc}>
    {/* Your App */}
  </RainbowKitProvider>
);
```

### 但它是什麼？

當從錢包列表中選擇一個錢包時，會有表情符號爆炸 🌈🌈🌈。 它使用你選擇的錢包圖標。
