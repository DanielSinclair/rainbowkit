# 鏈

## 自定義鏈

RainbowKit 是設計與 [wagmi 的 `chain` 對象](https://wagmi.sh/react/chains) 集成的。 請查看[這裡](https://wagmi.sh/react/chains#supported-chains) 支援的鏈清單。

### 自定義初始鏈

預設情況下，RainbowKit 將連接到 Wagmi 提供的第一個鏈。 此行為可以通過 `initialChain` 屬性來自定義。

可以使用鏈 ID 來配置初始鏈。

```tsx
<RainbowKitProvider initialChain={1}>
```

為了方便起見，您還可以傳遞一個鏈對象。

```tsx
<RainbowKitProvider initialChain={mainnet}>
```

### 自定義鏈元數據

默認情況下，會提供幾個鏈圖標和背景，但您可以使用 `iconUrl` 和 `iconBackground` 屬性來自定義每個鏈的圖標和背景。

使用 `getDefaultConfig` 的範例

```tsx
import { mainnet, optimism } from 'wagmi/chains';

const config = getDefaultConfig({
  chains: [
    {
      ...mainnet,
      iconBackground: '#000',
      iconUrl: 'https://example.com/icons/ethereum.png',
    },
    {
      ...optimism,
      iconBackground: '#ff0000',
      iconUrl: 'https://example.com/icons/optimism.png',
    },
  ],
});
```

使用 `createConfig` 的範例

```tsx
import { mainnet, optimism } from 'wagmi/chains';
import { Chain } from '@rainbow-me/rainbowkit';

const chains: readonly [Chain, ...Chain[]] = [
  {
    ...mainnet,
    iconBackground: '#000',
    iconUrl: 'https://example.com/icons/ethereum.png',
  },
  {
    ...optimism,
    iconBackground: '#ff0000',
    iconUrl: 'https://example.com/icons/optimism.png',
  },
];

const config = createConfig({
  chains,
});
```
